# 📈 Trader Performance vs. Market Sentiment Analysis
**Data Science & Analytics Intern – Round 0 Assignment**

---

## 📌 Project Overview
This project investigates the relationship between trader performance (PnL) and market sentiment, using the **Fear and Greed Index**. By integrating a massive dataset of over 211,000 historical trade records with sentiment classifications, I identified how different trader profiles respond to market extremes and developed data-driven strategies for risk management and capital allocation.
